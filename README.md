Bergen Tech GPA Calculator
======================================

This is a maven project, to build it:

    $ mvn clean package

To run the example `org.eclipse.jetty.demo.Main`:

    $ mvn exec:exec

Open your web browser to:

    http://localhost:8080/  

To stop Jetty:

  use <kbd>CTRL</kbd>+<kbd>C</kbd>
